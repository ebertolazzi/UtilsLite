Detailed Description

Queue for functions with signature void()

This queue is dependent on template parameter Function. Only one type of functions can be queued. For example, if this queue is instantiated for a lambda function, only this exact lambda function can be queued, since each lambda function has its own type separate from all other lambda functions.

To make this queue more flexible, instantiate it either with std::function<void()>, or use the virtual thread pool VirtualThreadPool.
