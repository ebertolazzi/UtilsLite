Detailed Description

Implementation of thread pool.

This class manages a pool of threads that can execute tasks concurrently.

void Utils::ThreadPool2::join ( )

Discard all tasks from the queue that have not yet started and wait for all threads to return. Also throws an exception if one of the tasks has encountered an uncatched exception. Leaves the pool in a shutdown state not ready to run tasks, but ready for destruction.
