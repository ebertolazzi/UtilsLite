42 // HJPatternSearch class decription
43 //
44 // This class implements the Hooke-Jeeves Pattern Search Algorithm.
45 // The references can be found in:
46 //
47 // R. Hooke, T. A. Jeeves,
48 // "Direct Search" Solution of Numerical and Statistical Problems,
49 // Westinghouse Research Laboratories,
50 // Pittsburg, Pennsylvania
51 //
52 // Arthur Kaupe,
53 // Algorithm 178: Direct Search,
54 // Communications of the ACM,
55 // Volume 6, Number 6, June 1963, page 313.
56 //
57 // M Bell, Malcolm Pike,
58 // Remark on Algorithm 178: Direct Search,
59 // Communications of the ACM,
60 // Volume 9, Number 9, September 1966, page 684.
61 //
62 // FK Tomlin, LB Smith,
63 // Remark on Algorithm 178: Direct Search,
64 // Communications of the ACM,
65 // Volume 12, Number 11, November 1969, page 637-638.
66 //
67 */
